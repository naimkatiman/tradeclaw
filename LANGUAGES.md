## Contributing a Translation

Want to translate TradeClaw into your language? We'd love your help!

1. Copy `README.md` as `README.xx.md` (where `xx` is the [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) language code)
2. Translate all prose, headers, feature descriptions, and CTA text
3. **Keep all badge image URLs and links in English** (they must work)
4. **Keep code blocks in English** (commands don't change)
5. Add a language switcher row at the top linking to all existing translations
6. Update this file (`LANGUAGES.md`) with your new language
7. Update `README.md` to include a link to your translation in the language row
8. Open a PR!

### Translation Guidelines

- Use culturally appropriate phrasing — don't translate literally
- Technical terms (e.g., RSI, MACD, Docker) stay in English
- Keep the same document structure as the English version
- Test that all links work correctly
