## Adding a New Translation

We welcome translations into any language! To contribute:

1. Fork the repository
2. Copy `README.md` to `README.<lang-code>.md` (e.g. `README.es.md` for Spanish)
3. Translate the content — keep code blocks and commands in English
4. Add your language to this file and to the language row in `README.md`
5. Open a pull request with the title: `docs: add <Language> translation`

### Translation Guidelines

- Keep **all code blocks** exactly as-is (commands, YAML, HTML snippets)
- Keep **technical terms** in English: RSI, MACD, EMA, Docker, npm, API, CLI, Webhook
- Use **natural, idiomatic** language — not word-for-word translation
- Translate **headings, descriptions, feature bullets, and comparison table text**
- Mirror the structure and sections of the English README
