# TradeClaw Learned Rules

These are the current learned operating rules produced by the trust-loop reflection process.

## Current rules
- Prefer one verified structural improvement over multiple speculative edits.
- Prioritize real signal reliability and proof quality before marketing polish.
- Do not make stronger public claims than the tracked evidence supports.
- When conversion is weak, inspect funnel clarity and checkout reliability before adding new copy.
- When signal quality is weak, bias toward suppressing low-quality setups before expanding coverage.
