## Description

Adding [TradeClaw](https://github.com/naimkatiman/tradeclaw) to the list of open-source algorithmic trading tools.

## About TradeClaw

TradeClaw is a self-hosted AI trading signal platform that traders and developers can run on their own infrastructure:

**Signal Generation**
- Multi-indicator confluence: RSI, MACD, EMA-20/50/200, Bollinger Bands
- Confidence scoring (0–100%) with threshold filtering
- Supports 15+ assets: forex (EURUSD, GBPUSD, USDJPY), metals (XAUUSD, XAGUSD), and crypto (BTCUSD, ETHUSD)

**Backtesting**
- Test strategies against historical OHLCV data
- Metrics: win rate, profit factor, max drawdown, Sharpe ratio
- Configurable date ranges and position sizing

**Paper Trading**
- Simulated trading with virtual capital
- Tracks open positions, P&L, and trade history

**Infrastructure**
- One-command Docker Compose deploy
- REST API with full OpenAPI 3.0 spec
- Webhook delivery to any endpoint
- Telegram bot integration

- **GitHub:** https://github.com/naimkatiman/tradeclaw
- **Demo:** https://tradeclaw.win
- **License:** MIT
- **Stack:** Next.js 15, TypeScript, Node.js
